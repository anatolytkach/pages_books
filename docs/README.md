This directory is limited to operational documentation that is still required
for the current repository.

Repository startup context does not live here. Start with the root files:

- `PROJECT_CONTEXT.md`
- `PROJECT_RULES.md`
- `CURRENT_STATE.md`
- `COMPONENT_GUIDE.md`
- `AGENTS.md`

## Deliberate Scope Rule

This directory should contain only documentation that is required to operate or
verify the current project tooling.

It should not contain:

- architecture history
- phase logs
- cleanup diaries
- parity journals
- obsolete exploratory notes
